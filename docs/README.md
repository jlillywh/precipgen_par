# PrecipGen PAR Documentation

This directory contains all documentation for the PrecipGen PAR project.

## Directory Structure

User-facing documentation and tutorials:

- `QUICK_REFERENCE.md` - Command reference and examples
- `STREAMLIT_GUIDE.md` - Web interface documentation
- `STREAMLIT_QUICKSTART.md` - Quick start for web interface

- `STREAMLIT_README.md` - Comprehensive Streamlit docs
